# Clock In Clock Out

A simple work timer and billing tracker for freelance development sessions.

## Project Purpose

Clock In Clock Out is a lightweight application for tracking freelance development work sessions. The project is designed to help developers record billable time, organize work by project or client, and maintain a clear history of completed work.

The goal is to build a practical portfolio-ready project that demonstrates backend development, data handling, session tracking, documentation, and eventual dashboard/reporting features.

## Planned Features

- Start and stop work sessions
- Track active and completed sessions
- Add session notes
- Associate sessions with projects or clients
- Calculate total billable time
- Estimate invoice totals
- Review work history
- Export records for billing or documentation
